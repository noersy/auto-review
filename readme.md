# Auto Review Bot

Auto Review PR using Jenkins and AI code assistants.

## Description

This project automates the review of Pull Requests (PRs) by utilizing Jenkins and Claude Code CLI (or other configured AI tools) to analyze changes and provide automated feedback.

## Installation

1. Clone the repository
2. Install dependencies:

   ```bash
   npm install
   ```

3. Set up your `.env` file based on the required configurations.

## Usage

Start the bot locally:

```bash
npm start
```

Or run via the provided CLI:

```bash
auto-review
```

## Testing

Run the test suite using Jest:

```bash
npm test
```
